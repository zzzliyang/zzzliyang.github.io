About

In this post I will show the steps of a quick Spring Boot web application :)

Generate Spring Boot starter project

Go to Spring Boot Starter.

Click “Generate Project” to download Spring Boot project skeleton. Unzip and import project to Intellij.

These “starters” in build.gradle are important in Spring Boot. It allows you to define “one” dependency which implicitly requires all other necessary dependencies.

BlogDemoApplication is the starting point of this Spring Boot project.

Home Page

Add an index.html in resources/static/ folder. This is the default welcome page.

If you start BlogDemoApplication.main(), and go to localhost, you will be able to see your web app up and running!

Serving static files

With the help of Thymeleaf, you can serve static html files using controller. All html files should be placed in resources/templates/ folder.
